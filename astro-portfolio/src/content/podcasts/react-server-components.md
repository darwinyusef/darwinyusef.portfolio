## Descripción del episodio

Exploramos React Server Components, una de las innovaciones más importantes en el ecosistema de React.

### Temas tratados

1. **¿Qué son Server Components?**
   - Client vs Server Components
   - Diferencias con SSR tradicional
   - Ventajas de performance

2. **Patrones de Uso**
   - Cuándo usar Server Components
   - Composition patterns
   - Data fetching en el servidor
   - Streaming y Suspense

3. **Next.js App Router**
   - Migración desde Pages Router
   - Server Actions
   - Parallel routes y intercepting routes
   - Metadata API

4. **Optimizaciones**
   - Reducción del bundle de JavaScript
   - Mejora de Web Vitals
   - SEO benefits
   - Edge runtime
